== Installation ==

Starting with Visidea consists of just two steps: installing and setting up the plugin.

### INSTALL VISIDEA MANUALLY

1. Upload `visidea/` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Create an account on visidea.ai and copy/paste the tokens in the configuration page

### Updating ###

Automatic updates should work smoothly, but we still recommend you back up your site.
